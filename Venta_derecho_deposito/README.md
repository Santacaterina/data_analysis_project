**Limpieza y Optimización de Datos De Clientes contactados en Campaña de Venta de Certificado de Depósito a Término**

Una institución financiera externalizó una campaña de telemarketing para vender depósitos a plazo.
El objetivo de este proyecto era evaluar no solo las tasas de conversión, sino también la rentabilidad real de la campaña.

**Objetivo**
Transformar un conjunto de datos de marketing tradicional en un modelo de rendimiento financiero mediante:

**Cálculo del coste operativo por llamada**
Estimación del beneficio por cliente
Medición del ROI
Simulación de escenarios de optimización

**Dataset**
45215 registros
Variables demográficas, financieras y de campaña
Depuración y validación con Python (Pandas)

**Metodología**
Depuración y validación de datos
Tratamiento de valores atípicos
Ingeniería de características (coste, beneficio, ROI)
Análisis de rentabilidad por segmentos
Simulación de escenarios para la optimización de costes

**Información clave**
El ROI disminuye drásticamente después del tercer intento de contacto.
Los clientes con saldo negativo generan pérdidas estructurales (excepto en segmentos específicos).
Se detectó una estacionalidad significativa.
Un modelo de optimización híbrido reduce los costos operativos en aproximadamente un 30%, manteniendo la escalabilidad.

**Herramientas utilizadas**
Python (Pandas)
Google Colab
Looker Studio (panel de control de 3 páginas + portada)

**Resultado estratégico**
Se propuso un modelo operativo híbrido que equilibra la eficiencia y la cobertura del mercado, demostrando cómo el análisis de datos puede respaldar directamente la toma de decisiones financieras.
